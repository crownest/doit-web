# Installation the packages


**Node.js**
```bash
apt-get update
apt-get install nodejs
```

**Npm**
```bash
apt-get install npm
```

**Ruby**
```bash
apt-get update
apt-get install ruby-full
```

**Sass**
```bash
gem install sass
```

**Grunt**
```bash
npm install -g grunt
```

**Note**: If run npm command gives error "/usr/bin/env: node: No such file or directory"
```bash
ln -s /usr/bin/nodejs /usr/bin/node
```
